
## 如何运行：训练 & 测试

根据项目分析，这是一个**排序学习选股系统**，使用 StockTransformer 模型对沪深300成分股进行排序并输出 Top5。

---

### 一、环境准备

```bash
# 1. 进入项目目录
cd /Users/conrad/Desktop/THU-BDC2026-main

# 2. 安装 uv 包管理器
pip install uv

# 3. 安装依赖
uv sync

# 4. 安装 TA-Lib C库（必需！）
#    macOS:
brew install ta-lib
#    或从源码编译，参考 README.md
```

---

### 二、数据准备

```bash
# 下载股票数据（修改 get_stock_data.py 第223-224行的日期范围）
python get_stock_data.py

# 划分训练集/测试集（按日期）
python data/split_train_test.py
```

---

### 三、训练

```bash
# macOS / Linux
sh train.sh

# Windows 或直接运行
python code/src/train.py
```

训练过程中会自动：
- 划分训练集/验证集（最后2个月为验证集）
- 使用 `WeightedRankingLoss`（listwise + pairwise 组合损失）
- 每个 epoch 计算 `final_score`，保存最优模型到 `model/60_158+39/`
- 输出 TensorBoard 日志

当前已有训练好的模型：`model/60_158+39/best_model.pth`（final_score = 0.037838）

### 七、可调参数（`code/src/config.py`）

| 参数 | 默认值 | 说明 |
|---|---|---|
| `sequence_length` | 60 | 回看窗口（交易日数） |
| `feature_num` | 158+39 | 特征数量（可改为 39） |
| `d_model` | 256 | Transformer 嵌入维度 |
| `batch_size` | 4 | 批次大小 |
| `num_epochs` | 50 | 最大训练轮数 |
| `learning_rate` | 1e-5 | 学习率 |

如果显存不够，可以将 `d_model` 调小（如 128），或减少 `num_layers`（如改为 2）。
